# 도면·거래 통합 업무시스템

도금·표면처리 업체의 도면 등록부터 단가 승인, 주문·납품, 거래명세서,
매출 분석과 CS까지 하나의 흐름으로 연결하는 데스크톱 웹 애플리케이션입니다.

## 현재 구현 범위

- Notion 스타일의 데스크톱 운영 대시보드
- 도면·품목 목록, 검색, 상태 필터와 상세 검토 화면
- 신규 도면 등록 및 인식 대기 흐름(현재는 데모 시뮬레이션)
- 형상·면적·도금 난이도 기준의 단가 산정 내역과 승인 흐름
- 주문·납품 상태, 거래명세서 미리보기와 발급 처리
- 고객사 Excel(`.xlsx`) 원본 업로드, 열 매핑, 행 검증과 중복 제외
- Excel 등록 이력·원본 다운로드와 등록 거래 기반 매출 집계
- 월별·연도별 매출 요약, 고객사별 매출
- CS·클레임 칸반 및 처리 완료 흐름
- 고객사와 영구 저장되는 시스템 설정 화면

시스템 설정과 Excel 거래 데이터는 D1에, 업로드한 Excel 원본은 R2에 저장됩니다.
도면 OCR, 텔레그램 봇, 회계 전송은 다음 백엔드 단계에서 연결합니다.
도면 등록·승인·발급 상태 변경은 현재 브라우저 내 데모 데이터입니다.

## 실행

Node.js 22.13 이상과 pnpm이 필요합니다.

```bash
pnpm install
pnpm dev
```

검증:

```bash
pnpm test
```

## 기술 구성

- Next.js 16 / React 19
- vinext / Cloudflare Workers
- Cloudflare D1 / R2
- Sites 배포
